CG Programs

Here are some key programs I created that demonstrate CG comcepts. This includes lighting, and geometric algorithms.

Spring 2020

Rotating Cubes!

The very project that the background image is based on! This project gave me experience in sending data to a GPU once, and then dynamically using it as many times as we like based on arbitrary user input! This program also implements somewhat randomized rigid body transforms

Spring 2020

Introduction to Lighting

This is a watershed project for me. It was very challenging to understand how a lighting model works on 3D geometry, but once it clicked, I was never the same! Math and coding coincided in such a wonderful way that it made me want to pursue graphics programming professionally! This is an implementation of the Phong Illumination Model (not to be confused with the Phong shading model or per-fragment shading)

Spring 2020

Introduction to 3D Animation Programming

This project was downright fun! It's a basic demonstration of a looping animation on a segmented 3D model.

Fall 2020

2D Character Made From Bezier Polygons

This was a fun project in which I used manually programmed the algorithms required for cubic Bezier polygons to make a 2D simple character. This Mathematica notebook also has an interactive section in which the tangent line is calculated at points all around the Bezier character.

Fall 2020


3D Bezier Surfaces

This project involved creating several types of 3D surfaes using my implementation of the Coons method to interpolate interior Bezier control points given an outer skeleton of points. Additionally, I had fun rendering Bezier curves on the plane, and then calculating tangent lines per-point on the curve.

Fall 2020

3D B-spline implementation

Here, I used 3D B-splines to make pretzel-like rings! It was fun!

Fall 2021

Per-Fragment Shading

This project was largely review on per-vertex lighting (see Introduction to Lighting), but I did get to implement per-fragment lighting in addition! While such a prospect is not inherently realistic, I think it looks pretty great! (Note, the base canvas and models used were provided to me)

Fall 2021

Depth-Oriented Shading

I learned a lot about how depth-related details can enhance a scene. It was a ton of fun replicating commonly-seen effects! In addition to 3D anaglyph rendering, I also implemented foveated and depth of field rendering.

Fall 2019

Semantics & Syntax Verification

Here is my contribution towards a larger coding project that takes input of a simplified fictitious programming language, and tells the user if they had a syntax or semantic error, where it is in their input, and what kind it was precisely. It may be hard to understand out of context, but my contribution is everything besides the lexer, debugging tools, and input buffer.

Fall 2019

A Mini Compiler

This was a fascinating project for me that involved taking in input of a made-up programming language, and compiling it (converting it) to a list of instructions in a "lower level" format which is analagous to how real compilers generate assembly code.

Miscellaneous Passion projects

Related to coding or not, here are some of my labors of love that do not neatly fall into a category. Hope you find something in them!

Spring 2020

Tethys: My Peak into game development

Tethys is a small science fiction game I created in a week long game jam-like environment. It serves more of a proof of concept, as I worked on this alone, and there are still bugs. It was a lot of fun! Do note that, according to the project specifications, the 1 2 and 3 keys get you to different areas of the game, C gets you to the end screen, shift is to attack, space is to jump, arrows move, and M gets you to the menu on which you use the mouse.
